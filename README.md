# PathForge 🚀
> Forge Your Path to Your Dream Role

PathForge is an AI-powered career navigation platform that helps college
students get from where they are to where they want to be — with
personalized roadmaps, skill gap analysis, and ML-based readiness scoring.

---

## ✨ Key Feature
**Dream Role Reverse Planner** — Enter your dream company and role.
PathForge reverse-engineers your exact personalized roadmap to get there,
recalibrated every week based on your progress.

---

## 🛠️ Tech Stack
- **Frontend** — React.js + Tailwind CSS
- **Backend** — Node.js + Express.js
- **Database** — MongoDB Atlas
- **AI** — Gemini API
- **ML** — Python + FastAPI + scikit-learn
